******************************************
EventTracking for StatPress
******************************************

EventTracking for StatPress)は以下の環境で動作します。

Wordpress 4.1 以上 
PHP 5.3
MySQL 5.0
文字コード UTF-8


【インストール】
解凍して得られたフォルダ「joc-dl-checker-management」をWordpressのpluginsフォルダの中に設置してください。
アップデートする場合は「joc-dl-checker-management」を上書きします。
自動アップグレードの案内が出ているときは、一旦joc-dl-checker-managementを停止してからアップグレードして
ください。停止せずに自動アップグレードを行った場合は、アップグレード後に停止と有効化を
行ってください。

いずれの場合も本体へのカスタマイズは上書きされてしまいますのでご注意ください。

【インストールするときの注意】
不測の事態を避けるためにも EventTracking for StatPress を
有効化する前に既存のブログのバックアップを取ってください。

JavaScript を多用しているテーマは利用できない場合が有ります。
また、他のプラグインとの競合で動作不良を起こす場合も有ります。
動作がおかしい場合は他のプラグインを停止し、動作確認してください。

EventTracking for StatPressを使用中に起こった如何なる損害も自己責任となります。




【開発環境】

OS linux
PHP 5.3.3
MySQL 5.0
